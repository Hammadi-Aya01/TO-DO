# ✅ My To-Do App

A simple To-Do List web application that helps users add and manage daily tasks.

---

## ✨ Features

- Add new tasks
- Display task list
- Empty list message
- Simple and clean interface

---

## 🛠️ Technologies

- HTML
- CSS
- JavaScript

---

## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/Hammadi-Aya01/TO-DO.git
```

Open the file:

```bash
todo.html
```

You can run it directly in any web browser.
